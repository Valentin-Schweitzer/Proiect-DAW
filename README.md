# Titlu: Biblioteca Online – platformă web pentru gestionarea și împrumutul de resurse digitale

## Descriere generală

**Biblioteca Online** este o aplicație web care permite utilizatorilor să exploreze, să rezerve și să împrumute documente dintr-o bibliotecă digitalizată.  
Aplicația are rolul de a moderniza activitatea unei biblioteci publice și de a oferi acces facil la colecții de cărți, reviste și articole – atât în format fizic, cât și digital.

Platforma este construită pe arhitectura clasică **PHP + MySQL**, cu o interfață prietenoasă realizată în **HTML**, **CSS** și **JavaScript**.  
Utilizatorii se autentifică pe site, pot căuta documente, pot face rezervări, iar bibliotecarii gestionează împrumuturile și actualizează stocul de exemplare.
